🚀 Smart Features Demo

✨ Welcome to the Smart Website!

This page demonstrates all the intelligent features that make this website modern and user-friendly. Scroll down to explore each feature!

Reading Progress Bar Dark Mode Keyboard Shortcuts Smart Navigation

📊 Reading Progress Bar

Look at the top of your screen! You'll see a colorful gradient bar that shows your reading progress.

How it works:
  • Automatically tracks your scroll position
  • Shows percentage of page scrolled
  • Beautiful animated gradient effect
  • Works on all pages

Try scrolling up and down to see it in action!

🌓 Dark Mode Toggle

Look at the bottom right corner! You'll see a sun/moon button.

Features:
  • One-click dark/light mode switching
  • Saves your preference automatically
  • Smooth color transitions
  • Eye-friendly for night reading

👉 Try it now! Click the button in the bottom right corner to switch themes.

Your preference is saved, so the next time you visit, your chosen theme will be active.

⌨️ Keyboard Shortcuts

Power users rejoice! Navigate faster with keyboard shortcuts.

Available shortcuts:
  • / - Focus the search bar
  • h - Go to home page
  • t - Scroll to top
  • b - Scroll to bottom
  • d - Toggle dark mode
  • ? - Show/hide shortcuts help

👉 Press ? key right now to see a beautiful modal with all shortcuts!

Note: Shortcuts don't work when you're typing in input fields.

⏱️ Reading Time Estimator

This feature automatically calculates how long it will take to read an article.

How it calculates:
  • Counts all words in the article
  • Uses average reading speed (200 words/min)
  • Only shows for articles with 200+ words
  • Displayed in a beautiful gradient badge

This article is long enough, so you should see a reading time badge at the top!

📑 Auto Table of Contents

For longer articles with multiple sections, a table of contents is automatically generated.

This is a Sample H3 Heading

The table of contents feature detects all H2, H3, and H4 headings in your article.

Another Sample Section

Each heading becomes a clickable link in the TOC. When you click, it smoothly scrolls to that section.

Sample H4 Subsection

H4 headings are indented in the TOC to show hierarchy.

Smart features:
  • Auto-generates unique IDs for headings
  • Smooth scroll to sections
  • Highlights current section while scrolling
  • Shows hierarchical structure
  • Only appears for articles with 3+ headings

You should see a TOC box near the top of this page!

📋 Copy Code Button

Any code block on the site now has a convenient copy button.

Example code block:

function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to the smart website!`;
}

// Try hovering over this code block
greet('Developer');

👉 Hover over the code above - you'll see a "Copy" button appear in the top right!

Click it to copy the code to your clipboard. You'll get visual feedback when copied.

👉 Scroll down more and look at the bottom right corner to see it!

🔗 Smart External Links

All external links are automatically enhanced for better UX and security.

Example external links:

  • GitHub - Opens in new tab
  • Google - Opens in new tab
  • Stack Overflow - With security attributes

External links automatically get:

  • target="_blank" to open in a new tab
  • rel="noopener noreferrer" for security
